API Usage¶

The purpose of this example is to illustrate how to use the API. For practically, we are using the https://lbrack.testspace.com/ organization which doesn’t have any read restrictions.

[1]:

from IPython.display import display # Pretty display in Python
import testspace_colab
testspace_colab.use_test_config()      # Comment this line if you want to use you own config.
# testspace_colab.use_samples_config() # uncomment this line to use the testspace samples
api = testspace_colab.API()

using TS_COLAB_CONFIG_DIR=/home/laurent/github/laurent/testspace-colab/tests/.config/test

You can call any API provided by the testspace-python as illustrated below

The following example shows how to use the API in a project. If you know the json structure, you can use the jsonpath-ng module to extract any information you are interested in usig the jsonpath notation. For those of us familiar with XPATH, check the JSONPath - XPath for JSON

In the example below, we extract the list of test cases associated to that result

[3]:

import jsonpath_ng
details = api.get_result_details(result=last_result['id'], project=PROJECT, space=SPACE)
[match.value for match in jsonpath_ng.parse('$..cases[:]').find(details)]
