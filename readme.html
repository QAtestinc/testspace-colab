testspace-colab¶

A collaborative environment to experiment with data extracted from testspace. This project provides a complete environment to mind data from Testspace, persist it, and analyze it. The intent is to provide a prototyping environment for Testspace’s Insight 2.0.

  • Free software: MIT license

  • Documentation: documentation

Features¶

  • Contains testspace client binaries (2.5.4061)

  • Provides a CLI (ts-colab) as well as an API

  • See the user manual to get started

  • Provides facility to start an ELK (ElasticSearch/Logstach/Kibana) via the sebp/elk docker image. All that is needed is to have docker installed on your machine. Refer to the documentation for additional details.
